# music-streaming-db
# Music Streaming Database

A relational database project built with PostgreSQL modeling a music streaming platform similar to Spotify.

## Database Schema
8 tables: artists, albums, songs, users, streams, playlists, playlist_songs, subscriptions

## Analytical Queries
1. Top 10 most streamed songs
2. Stream completion rate by artist
3. Premium vs free user streaming behavior
4. Streaming activity by device type
5. Streaming trends by day of week
6. Top genre by stream count

## Tools Used
- PostgreSQL
- DataGrip
- Python (pandas) — in progress

## Key Findings
- Hip-Hop is the most streamed genre with 46 streams
- Mobile is the most popular device but tablet users have the highest completion rate
- Premium users stream more and complete more songs than free users
- Monday is the most active streaming day
